# Street Crashers (Pyxel)

A small 2D fighting game prototype made with **Pyxel**.

## How to run
```bash
pip install pyxel
python main.py
```

## Controls
### Menus
- **ENTER**: Start / Confirm
- **ESC**: Quit

### Character Select
- **P1**: `A/D` to choose, **ENTER** to lock
- **P2**: `LEFT/RIGHT` to choose, **ENTER** to lock

### Stage Select
- `A/D` or `LEFT/RIGHT` to choose stage
- **ENTER** to begin

### Fight
- **P1**: `A/D` move, `W` jump, `S` crouch, `J` punch, `K` kick, `U` special
- **P2**: `LEFT/RIGHT` move, `UP` jump, `DOWN` crouch, `N` punch, `M` kick, `,` (comma) special

First fighter to reduce the other’s HP to 0 wins.
